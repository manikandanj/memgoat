MemGoat

You wake inside a witch-worked ruin with your short-term memory collapsing every two minutes.

You do not see yourself. The story is told from your point of view. Inspect rooms, choose which memories the Cave Echo should keep, and carry those memories through sealed rooms.

Core mechanic

Inspections reveal candidate memories. Committed memories survive resets and unlock backend room progression.

Prototype goal

Move from the Waking Chamber to the Bell Gallery and Root Gate, then ask the Echo the final question.

Controls

Hover objects to highlight them. Click to inspect. Open the Echo Graph to review memory state and cross thresholds.

Integrated state

This build keeps the new static presentation while using FastAPI as the source of truth.
