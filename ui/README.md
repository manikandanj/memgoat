# Goat of the Witch-Cave

Static frontend prototype based on the provided sketch.

Open `index.html` in a browser.

## Features
- Clue Box modal with found clues and connections.
- 2-minute memory timer that wipes current visible clues.
- Cave Voice side chat that gives indirect hints based on previously discovered clues.
- Three sample dungeon rooms.
- No-backtracking rule after moving forward.
- Left/right arrow buttons and keyboard arrow support.
- Bottom narration box for goat thoughts.

This is pure HTML, CSS, and JavaScript. No backend or API key required.
